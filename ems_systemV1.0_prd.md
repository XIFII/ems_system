## 1. 背景介绍

### 1.1 业务背景

园区级微电网 EMS 需要在本地边缘侧完成源网荷储充协同控制。策略中心负责配置运行模式、硬性约束、动环联锁策略以及 L3 智能调度策略；电价配置负责提供削峰填谷、经济调度、收益测算的基础价格数据。

当前产品定位为边缘 EMS 系统，本地控制优先，云端仅作为可选辅助能力。本 PRD 聚焦当前 Demo 已实现或本轮新增的两类页面：

- 策略中心
- 电价配置

### 1.2 业务场景

| 场景 | 目标 | 主要功能 |
| --- | --- | --- |
| 运行模式切换 | 允许运维人员明确 EMS 的就地/远方控制权 | 本地自治、云端辅助、电网调度等模式选择 |
| 硬性约束保护 | 在边缘侧保障并网合规与变压器安全 | 防逆流、需量控制启停与参数配置 |
| 动环策略 | 将 DI/DO 采集器、设备状态与 EMS 控制动作联动 | DI/DO 点位命名、动环策略条件/动作配置 |
| 智能调度 | 通过模板和策略分配实现削峰填谷等调度 | 月默认模板、日覆盖、批量复用弹窗、充放电模板、曲线预览、智能规划 |
| 固定分时电价 | 支撑长期稳定的 TOU 电价策略 | 尖/峰/平/谷/深谷时段与价格维护 |
| 动态市场化电价 | 支撑日前市场化价格曲线录入 | 48 点或 96 点价格表、按日历选日、导入、折线预览 |

## 2. 产品概述

### 2.1 产品定位

策略中心与电价配置是边缘 EMS 的核心配置模块。策略中心决定 EMS 如何响应现场运行状态与调度目标；电价配置为削峰填谷和经济调度提供价格依据。

### 2.2 服务对象

| 角色 | 使用目标 |
| --- | --- |
| 园区能源主管 | 查看策略启停状态，确认削峰填谷策略应用关系与电价配置。 |
| 运维工程师 | 配置 DI/DO 点位名称、动环联锁策略、防逆流、需量控制等本地策略。 |
| 调度/算法工程师 | 配置 L3 模板、检查充放电曲线、录入市场化电价数据。 |
| 测试/交付工程师 | 验证策略配置是否满足现场联调、验收和安全约束。 |

### 2.3 产品逻辑

1. 感知：通过设备物模型、电表、PCS、BMS、空调、DI/DO 采集器获取实时状态。
2. 判断：L1/L2/L3 与动环策略根据状态、阈值、电价和模板生成执行意图。
3. 执行：EMS 将策略转换为设备动作、DO 输出或平台动作。
4. 反馈：页面展示策略启停、模板曲线、电价曲线、导入状态和配置校验。

### 2.4 功能清单

| 一级模块 | 二级模块 | 功能说明 |
| --- | --- | --- |
| 策略中心 | 模式切换 L2 | 配置 EMS 运行模式与控制权状态。 |
| 策略中心 | 策略监控 | 展示 L0-L3 当前已启用或正在运行策略的状态与日志。 |
| 策略中心 | 硬性约束 L1 | 配置防逆流、需量控制启停和简化参数。 |
| 策略中心 | 动环策略配置 | 配置 DI/DO 点位名称、动环策略条件和动作。 |
| 策略中心 | 智能策略 | 配置削峰填谷、智能经济调度、计划曲线、绿电消纳优先。 |
| 电价配置 | 电价模式设置 | 在固定分时电价与动态市场化分时电价之间选择当前唯一生效模式。 |
| 电价配置 | 固定分时电价 | 维护尖峰平谷深谷 TOU 模板。 |
| 电价配置 | 动态市场化分时电价 | 维护 48/96 点日前市场化电价曲线。 |

## 3. 系统与策略架构说明

### 3.1 策略层级

| 层级 | 名称 | 响应目标 | 页面映射 |
| --- | --- | --- | --- |
| L1 | 硬性约束层 | 秒级或更快的并网合规和容量保护 | 防逆流、需量控制 |
| L2 | 运行模式层 | 控制权与运行方式切换 | 模式切换 |
| L3 | 智能策略层 | 分钟级或计划级经济优化 | 削峰填谷、智能经济调度、计划曲线、绿电消纳 |
| L0/动环 | 环境与安全联锁 | 设备状态、DI/DO、消防与动环告警 | 动环策略配置 |

### 3.2 点位引用规则

DI/DO 点位必须使用“采集器设备 + 点位编码”的复合标识，不允许只使用裸 `DI1`、`DO1`。展示时使用：

- 所属采集器：例如 `EnvIO-A01`
- 点位编码：例如 `DI1`
- 点位名称：例如 `水浸`
- 物模型名称：例如 `water_leak_alarm`

### 3.3 电价数据规则

电价配置在同一时刻只允许存在一种当前生效模式：

- 固定分时电价：用于稳定 TOU 结构
- 动态市场化分时电价：用于日前市场曲线

动态模式支持：

- 48 点：每 30 分钟一个点
- 96 点：每 15 分钟一个点

动态电价允许负电价；固定分时电价不允许负电价。

### 4.1.3 策略监控

#### 页面目标

以 L3 到 L0 的动态约束拓扑展示当前已启用、已配置或正在运行策略的状态、最近动作和运行日志，帮助运维人员直观看到智能策略生成后如何被控制边界、硬性约束和动环联锁逐级约束，最终下发到设备执行对象。

#### 页面结构

| 区域 | 说明 |
| --- | --- |
| 动态层级拓扑 | 按 `L3 智能策略 -> L2 模式/并离网 -> L1 硬性约束 -> L0 环境联锁 -> 设备/执行对象` 展示运行链路。 |
| 拓扑节点 | 展示策略名称、运行状态、执行对象、优先级、最近动作、更新时间和说明。 |
| 层级汇总 | 点击层级标题时展示该层当前纳管策略数量、告警状态和链路刷新日志。 |
| 日志详情 | 点击策略节点后展示该策略的触发、执行、恢复、告警和人工操作日志。 |

#### 展示规则

| 规则 | 说明 |
| --- | --- |
| L3 | 作为策略生成源头，仅展示当前唯一运行的智能策略。 |
| L2 | 对 L3 生成的策略进行控制权和并离网边界校验。 |
| L1 | 对策略执行意图进行防逆流、需量控制和有序充电等硬性约束修正。 |
| L0 | 作为最终安全联锁层，仅展示已启用的动环策略，并可对执行命令进行拦截或放行。 |
| 设备/执行对象 | 展示 PCS、光伏、有序充电、可控负荷、QF/STS 等策略作用对象。 |
| 限制 | 当前 Demo 使用本地演示日志，不接真实策略事件流。 |

### 4.1.6 智能策略

#### 页面目标

提供边缘 EMS 的智能策略入口，包含削峰填谷、智能经济调度、计划曲线、绿电消纳优先。削峰填谷用于模板化充放电；智能经济调度用于历史/今日/未来日期查询、预测电价展示、系统自动生成日策略和人工调整；计划曲线用于接收用户或虚拟电厂调度计划，形成站级目标曲线，并分解到站内设备执行。

| 流程 | 选择日期 -> 计算中 -> 结果审核 |
| 结果 | 推荐时段列表、收益提示、策略曲线 |
| 底部按钮 | `取消`、`应用至当前模板` |

##### 应用规则

| 规则 | 说明 |
| --- | --- |
| 覆盖写入 | 应用后用规划结果整批覆盖当前模板时段 |
| 保留模板名称 | 不修改模板名称和模板 ID |
| 功率符号 | 充电为正，放电为负 |
| 合法性 | 规划结果必须满足模板保存校验 |

#### 页面结构

| 区域 | 说明 |
| --- | --- |
| 模式参数区 | 数据粒度、目标日期、平均电价、当前机制 |
| 日历选日区 | 按月展示每天的平均电价，支持点击日期切换日数据 |
| 导入与保存按钮 | 导入 CSV/Excel、保存数据 |
| 价格数据表格 | 逐行编辑市场电价 |
| 动态电价折线预览 | 展示市场电价折线和辅助底带 |
| 导入与保存状态 | 显示来源、时间、粒度、保存状态 |

#### 动态模式按钮

| 名称 | 位置 | 行为 | 二次确认 |
| --- | --- | --- | --- |
| 导入 CSV/Excel | 页面右上角 | 选择 `.csv`、`.xlsx`、`.xls` 文件并解析覆盖当前日期数据 | 文件覆盖前 V1 不二次确认，失败时提示 |
| 保存数据 | 页面右上角 | 校验当前数据并标记为已保存 | 否 |

| 下月 | 日历选日区 | 切换到下一个自然月 | 否 |

#### 动态输入框/下拉框清单

| 字段 | 必填 | 类型 | 长度/范围 | 默认值 | 联动/说明 | 校验 |
| --- | --- | --- | --- | --- | --- | --- |
| 数据粒度 | 是 | 下拉 | `96点/15分钟`、`48点/30分钟` | `96点/15分钟` | 切换后按日期加载对应粒度数据 | 条数必须严格匹配 |
| 目标日期 | 是 | date | 合法日期 | `2026-04-16` | 当前日前价格日期，与日历选中日期同步 | 浏览器原生日期校验 |
| 市场电价 | 是 | number | 允许负数，保留 4 位小数 | 演示价格 | 编辑后仅更新该日期、该粒度数据 | 必填，必须为数值 |

| 列 | 可编辑 | 说明 |
| --- | --- | --- |
| 序号 | 否 | 从 01 开始 |
| 开始时间 | 否 | 由粒度生成或导入 |
| 结束时间 | 否 | 由粒度生成或导入 |
| 市场电价(元/kWh) | 是 | 支持负值 |

#### 动态电价日历字段

| 字段 | 说明 |
| --- | --- |
| 星期表头 | 显示 `周一` 至 `周日` |
| 日期 | 当前月份内的自然日 |
| 节假日标签 | 使用本地演示数据标注法定节假日 |
| 平均电价 | 展示当日 48/96 点价格均值 |
| 当前标签 | 标识当前正在编辑的日期 |

#### 导入规则

| 项 | 规则 |
| --- | --- |
| 文件格式 | `.csv`、`.xlsx`、`.xls` |
| 条数 | 48 点模式必须 48 条，96 点模式必须 96 条 |
| 时间 | 必须覆盖 `00:00-24:00`，连续且不重叠 |
| 价格 | 必填，数值，允许负电价 |
| 导入结果 | 成功后覆盖当前表格数据，保存状态变为未保存 |

#### 日期与数据联动规则

| 场景 | 规则 |
| --- | --- |
| 点击日历日期 | 切换 `目标日期`，并加载该日对应粒度的价格数据 |
| 切换粒度 | 保持当前日期不变，切换到该日期对应粒度数据 |
| 编辑价格 | 仅修改当前日期、当前粒度下的数据 |
| 导入文件 | 仅覆盖当前日期、当前粒度下的数据 |
| 保存数据 | 仅保存当前日期、当前粒度下的数据状态 |

#### 保存校验

| 校验项 | 失败提示 |
| --- | --- |
| 条数不匹配 | 当前粒度要求 N 条数据 |
| 时间未覆盖全天 | 动态电价必须覆盖 00:00-24:00 |
| 时段不连续 | 动态电价时段必须连续且不重叠 |
| 价格为空或非数值 | 电价必须为数值 |

## 5. 非功能需求

### 5.1 性能要求

| 项 | 要求 |
| --- | --- |
| 页面首屏 | 本地部署环境下应在 3 秒内完成可交互渲染 |
| 动态 96 点表格 | 96 行编辑不应明显卡顿 |
| 模板曲线 | 切换模板后 500ms 内更新图表 |
| 导入解析 | 单日 48/96 点 CSV/Excel 文件应在 2 秒内完成前端解析 |

### 5.2 可用性要求

| 项 | 要求 |
| --- | --- |
| 只读与编辑安全 | DI/DO 默认只读，进入编辑后才允许修改 |
| 弹窗分层 | 模板弹窗与智能规划子窗口层级清晰 |
| 视觉边界 | 高频策略分配区置顶，低频模板配置区置底 |
| 状态反馈 | 保存、导入、校验错误必须在页面可见区域展示 |

### 5.3 安全与权限要求

| 项 | 要求 |
| --- | --- |
| 策略启停 | 后续接入真实下发时，应增加二次确认和操作记录 |
| 删除类操作 | 后续真实系统中删除时段、清除覆盖应记录操作人和时间 |
| 文件导入 | 仅解析本地 CSV/Excel，不执行文件内脚本或宏 |
| 数据下发 | 当前 Demo 不下发设备，真实版本需区分保存草稿与下发运行 |

### 5.4 限制需求

| 限制 | 说明 |
| --- | --- |
| 无后端持久化 | 当前版本为前端演示态，刷新后状态不保证保留 |
| 无真实物模型接口 | DI/DO 与设备点位使用本地模拟数据 |
| 无真实电价接口 | 动态市场化电价支持手工录入与导入，不自动拉取市场数据 |
| 无真实调度下发 | L1/L3 策略只做前端配置演示，不直接控制设备 |
